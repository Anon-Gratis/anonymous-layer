// ANON BROWSER pre-audit testnet · v0.0.0-alpha

Download

Anonymity-focused web browser for the Tor + anon-layer networks. Built on Mullvad Browser's hardened Gecko fingerprint baseline, with an anon-layer bridge and pre-installed extension that routes anon:// addresses through the project's hidden-service network.

PRE-AUDIT TESTNET — read the threat model first. This build is intended for protocol testing and feedback, not life-critical anonymity. The cryptographic core has not been independently audited; the hidden-service network ships only three directory authorities and a handful of relays. Threat model and current limits →

What you're installing

Each download is a self-contained directory or app bundle. Nothing is written outside it on install. Removal: delete the directory / drag the .app to the trash / scoop uninstall / brew uninstall --cask.

What's inside

  • Mullvad Browser ESR (Gecko engine) — hardened fingerprint, telemetry off, no Mozilla account integration.
  • anon-layer bridge — local HTTP gateway on 127.0.0.1:1081 that resolves anon:// URLs via 3-hop hidden-service circuits.
  • Bundled Tor — built-in (no separate install) for .onion access.
  • Bundled i2pd (Linux + Windows only) — for .i2p access, disabled by default.

What it does NOT do

  • No telemetry. No accounts. No phone-home updates (yet).
  • No automatic OS-level proxy changes. The browser tunnels its own traffic — other apps on your machine are unaffected.
  • Not signed for Windows SmartScreen / not Apple-notarized. Use the Scoop / Homebrew install paths to skip the warnings cleanly.

Reproducible builds against pinned upstream versions:

git clone https://github.com/anon-gratis/anonymous-layer

./browser-fork/scripts/repackage-mullvad.sh

./browser-fork/scripts/repackage-mullvad-windows.sh

# macOS (run on an actual Mac — uses hdiutil)
./browser-fork/scripts/repackage-mullvad-macos.sh
